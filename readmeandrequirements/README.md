# HI! WELCOME TO JABBERBOX!

*This is a Speech to Text to Speech program which consists of an input or microphone recording your voice, turning it into a transcript, and using that transcript by utilizing phonemes to create an output as an audio file, either to be saved or to be played right after and deleted in real time.*

This program can be used to create custom voices for speech augmentation, assistance, and the creation of a TTS audio that can replace a user's voice. It's also my final for **iDTech Teen Academy: NVIDIA AI and Machine Learning** course.
